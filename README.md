Dotfiles
========
Dotfiles for Mac using zsh and Vim.


Requirement
-----------
- Homebrew


Install
-------
``` shell
git clone https://github.com/hori-ryota/dotfiles.git ~/.dotfiles --recursive
cd ~/.dotfiles
./install.sh
```

### Optional

Install apps using Homebrew.
``` shell
cd ~/.dotfiles
./brew.sh
```
